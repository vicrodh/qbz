The Problem

Qobuz on Linux? It's complicated.

~

Web Player Caps Audio

Browser-based players limit output to 48kHz regardless of your subscription tier or source quality.

#

No Native App

Unlike Windows and macOS, Linux users have no official desktop application from Qobuz.

>

DAC Passthrough?

Getting bit-perfect audio to your DAC through a browser is practically impossible.

The Solution

QBZ delivers the experience you paid for.

✓ Native application with direct hardware access
✓ Full resolution support up to 24-bit/192kHz
✓ DAC passthrough for bit-perfect playback
✓ Open source, no telemetry, MIT licensed

Features

Everything you need, nothing you don't.

*

Hi-Fi Playback

  • Native FLAC decoding via Symphonia
  • DAC passthrough mode
  • Gapless playback ready
  • Quality selection with automatic fallback
  • Audio device enumeration and selection
/

Local Library

  • Scan and index your music collection
  • CUE sheet support for single-file albums
  • Metadata extraction (ID3, Vorbis, APE)
  • Embedded artwork extraction
  • FLAC, ALAC, WAV, AIFF, APE support
@

Integrations

  • MPRIS for Linux media keys
  • Last.fm scrobbling
  • Desktop notifications
  • SongLink/Odesli sharing
  • Chromecast support
=

Queue & Playlists

  • Full queue management
  • Shuffle and repeat modes
  • Qobuz playlists sync
  • Create and manage playlists
  • Favorites management

Built With

Frontend SvelteKit + TS
Backend Tauri 2.0 + Rust
Audio Rodio + Symphonia
Database SQLite
