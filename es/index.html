El Problema

¿Qobuz en Linux? Es complicado.

~

El Web Player Limita el Audio

Los reproductores basados en navegador limitan la salida a 48kHz sin importar tu suscripción o la calidad original.

#

Sin App Nativa

A diferencia de Windows y macOS, los usuarios de Linux no tienen aplicación de escritorio oficial de Qobuz.

>

¿Passthrough al DAC?

Obtener audio bit-perfect hacia tu DAC a través de un navegador es prácticamente imposible.

La Solución

QBZ entrega la experiencia por la que pagaste.

✓ Aplicación nativa con acceso directo al hardware
✓ Soporte completo de resolución hasta 24-bit/192kHz
✓ Passthrough DAC para reproducción bit-perfect
✓ Open source, sin telemetría, licencia MIT

Características

Todo lo que necesitas, nada que no.

*

Reproducción Hi-Fi

  • Decodificación FLAC nativa via Symphonia
  • Modo passthrough DAC
  • Reproducción gapless lista
  • Selección de calidad con fallback automático
  • Enumeración y selección de dispositivos de audio
/

Biblioteca Local

  • Escanea e indexa tu colección de música
  • Soporte CUE sheet para álbumes de un solo archivo
  • Extracción de metadatos (ID3, Vorbis, APE)
  • Extracción de artwork embebido
  • Soporte FLAC, ALAC, WAV, AIFF, APE
@

Integraciones

  • MPRIS para teclas multimedia de Linux
  • Scrobbling a Last.fm
  • Notificaciones de escritorio
  • Compartir via SongLink/Odesli
  • Soporte Chromecast
=

Cola y Playlists

  • Gestión completa de cola
  • Modos shuffle y repetición
  • Sincronización de playlists Qobuz
  • Crear y gestionar playlists
  • Gestión de favoritos

Tecnologías

Frontend SvelteKit + TS
Backend Tauri 2.0 + Rust
Audio Rodio + Symphonia
Base de datos SQLite
